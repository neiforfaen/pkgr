# pkgr

Interactive script picker for `package.json`. Detects your package manager and lets you fuzzy-filter scripts to run.

## Install

```sh
curl --proto '=https' --tlsv1.2 -LsSf https://github.com/neiforfaen/pkgr/releases/latest/download/pkgr-installer.sh | sh
```

## Usage

Run `pkgr` in any directory with a `package.json`.
